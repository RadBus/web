Welcome to RadBus!

For bus riders: thank you for using this free utility to track your bus schedule! 

If you have questions, issues, or comments, please visit our GitHub project to provide feedback. 

For programmers and anyone enthusiastic about open source software, we'd love to hear from you. RadBus is not one but two applicataions, the RadBus Web Client and the RadBus API.  The RadBus API helps apps to easily access Twin Cities Metro Transit departure data against a user's personalized schedule stored in the cloud.
